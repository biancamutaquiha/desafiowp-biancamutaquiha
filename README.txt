Configuração do ambiente para rodar a aplicação

Passo a passo:

0 - Abra o terminal;
1 - Instale o virtualenv. Exemplo: comando 'pip install virtualenv' ;
2 - Crie um ambiente virtual. Exemplo: comando 'virtualenv nome_do_ambiente' ;
3 - Ative seu ambiente virtual, passando no comando o path até o arquivo "activate". 
Exemplo: root/nome_do_ambiente/bin/activate
4 - Após ativar o ambiente virtual, navegue até pasta "desafiowp-biancamutaquiha" ;
5 - Instale os pacotes necessários para rodar a aplicação, a partir do arquivo "requirements.txt".
Exemplo: 'pip install -r requirements.txt' (aguarde instalação dos pacotes)
6 - Rode a aplicação com o comando 'python app.py' ;
7 - Abra o browser no endereço localhost na porta 5000 (http://127.0.0.1:5000/).
